# Slushi3_ChatBot
SLUSHI3_AI is a hardware-encrypted terminal that interfaces with the Gemini 2.5 Flash engine. Built on an ESP32 Quantum Core, it allows for real-time AI interaction through a web-based terminal and a physical SSD1306 OLED display. Designed for field operations, the unit is powered by a 1000mAh Li-Po cell with integrated TP4056 management.
